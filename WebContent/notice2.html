메인 화면입니다. 계절 별 대표적인 꽃을 확인 할 수 있으며 이 꽃들로 예쁜 꽃다발 역시 제작 가능합니다. 예약 관련은 [꽃다발 예약]을 참고해주세요.

[꽃다발 예약]

꽃다발을 에약할 수 있는 화면입니다. 지도를 통해 등록된 꽃 가게를 확인 후, 판매자 리스트에서 원하는 꽃 가게를 선택하여 주세요. 선택 후 예약하기 버튼 클릭 시, 메일 전송창으로 연결되게 됩니다. 메일로 예약 일자, 픽업 시간, 성함, 전화번호, 원하는 꽃다발 디자인을 포함하여 연락 넣어주시면 해당 판매자께서 확인 후에 답신 드릴 것입니다.

[원데이 클래스]

다양한 꽃집에서 진행되는 원데이 클래스를 소개합니다. 예약을 원할 시 꽃집으로 직접 연락해주시면 됩니다.

[공지사항]

다양한 공지를 포함하는 공간입니다. 더 궁금한 점 있으시면 관리자에게 문의 주세요.
